# Gameinyo
 
A survivors-like game built in Godot 4.6. Move, auto-attack, collect XP, level up, survive.
 
## Status
 
Early prototype. Core loop works — no upgrades, one enemy type, placeholder art.
 
## Run
 
Open `project.godot` in Godot 4.6.
 
## Controls
 
- WASD / Arrow keys — Move
